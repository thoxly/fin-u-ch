# Инструкция по восстановлению терминала

## Проблема

Терминал застрял в режиме ожидания незавершенной команды.
Prompt показывает: `cmdand cmdand cmdand cmdand while cmdand cmdand cmdand then dquote>`

## Решение

### Вариант 1: Прервать команду

1. Нажмите `Ctrl+C` 3-4 раза подряд
2. Если не помогает, нажмите `Ctrl+D`
3. Если все еще застряло, нажмите `Ctrl+Z`, затем выполните: `kill %1`

### Вариант 2: Закрыть и открыть новый терминал

1. Закройте текущий терминал
2. Откройте новый терминал
3. Перейдите в директорию проекта: `cd /Users/shoxy/Projects/fin-u-ch`

## После восстановления терминала
