# Индикатор загрузки банковских выписок

## Описание

Добавлен индикатор загрузки для процесса импорта банковских выписок, который информирует пользователя о процессе обработки больших файлов и не блокирует полностью UI.

## Изменения

### 1. Основной индикатор загрузки

При загрузке файла выписки на вкладке "Загрузка файла" отображается:

- **Оверлей с полупрозрачным фоном** с эффектом размытия (backdrop-blur)
- **Анимированная иконка загрузки** (вращающийся спиннер)
- **Информативные сообщения**:
  - "Загрузка выписки..."
  - "Пожалуйста, подождите. Обработка большой выписки может занять несколько минут."
  - "Вы можете свернуть это окно, загрузка продолжится в фоновом режиме."

### 2. Минимизированное окно

Когда окно импорта свернуто и идет загрузка:

- Отображается анимированная иконка загрузки
- Текст "Загрузка выписки..."
- Подсказка "Это может занять несколько минут"
- Кнопка закрытия заблокирована (`disabled`) во время загрузки

### 3. Зона drag-and-drop

Во время загрузки:

- Зона загрузки файлов становится неактивной (opacity: 50%)
- Курсор меняется на `cursor-not-allowed`
- Drag-and-drop события отключены
- Визуально понятно, что нельзя загрузить новый файл

## Технические детали

### Использованные компоненты

- `Loader2` из `lucide-react` - анимированная иконка загрузки
- Флаг `isLoading` из RTK Query мутации `useUploadStatementMutation`

### Стили

- Полупрозрачный оверлей: `bg-white/80 dark:bg-gray-900/80`
- Эффект размытия: `backdrop-blur-sm`
- Высокий z-index для оверлея: `z-10`
- Анимация вращения: `animate-spin`

## Преимущества

1. **Не блокирует пользователя** - окно можно свернуть, загрузка продолжится в фоне
2. **Понятная индикация** - пользователь видит, что процесс идет
3. **Информативность** - предупреждение о том, что загрузка может занять время
4. **Адаптивность** - работает как в развернутом, так и в свернутом состоянии
5. **Темная тема** - полная поддержка темной темы

## Пользовательский сценарий

1. Пользователь выбирает большой файл выписки (например, за год)
2. Появляется оверлей с индикатором загрузки
3. Пользователь видит сообщение, что процесс может занять несколько минут
4. Пользователь может продолжить работу:
   - Свернуть окно (загрузка продолжится)
   - Подождать на месте
5. После завершения загрузки открывается таблица маппинга с импортированными операциями
